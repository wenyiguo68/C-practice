题目：有一分数序列：2/1，3/2，5/3，8/5，13/8，21/13...求出这个数列的前20项之和。
```c
#include<stdio.h>

int main()
{
	float numerator = 2, denominator = 1;
	float sum = 0;
	for (int i = 0; i < 20; ++i)
	{
			float tmp = numerator / denominator;
			sum += tmp;
			int priorNumerator = numerator;
			numerator = numerator + denominator;
			denominator = priorNumerator;
	}

	printf("%f\n", sum);

	return 0;
}
```
Answer：
```
32.660263
```
