题目：809*??=800*??+9*?? 其中??代表的两位数, 809*??为四位数，8*??的结果为两位数，9*??的结果为3位数。求??代表的两位数，及809*??后的结果。
```c
#include <stdio.h>

int main()
{
	for (int i = 10; i < 100; ++i)
	{
		if (809 * i < 10000 && 809 * i>999)
		{
			if (8 * i < 100 && 8 * i>9)
			{
				if (9 * i < 1000 && 9 * i>99)
				{
					if (809 * i == 800 * i + 9 * i)
					{
						printf("%d\n", i);
						printf("%d\n", 809 * i);
					}
				}
			}
		}
	}

	return 0;
}
```
输出：
```
12
9708
```
