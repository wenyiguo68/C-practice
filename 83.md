题目：求0—7所能组成的奇数个数。
```c
#include <stdio.h>

int main()
{
	int sum = 0, cnt = 0;
	for (int i = 0; i < 8; ++i)
	{
		if (i == 0)
		{
			cnt = 4;
		}
		else if (i == 1)
		{
			cnt *= 7;
		}
		else
		{
			cnt *= 8;
		}
		sum += cnt;
		printf("%d位数为奇数的个数为%d\n", i + 1, cnt);
	}

	printf("总数为%d\n", sum);

	return 0;
}
```
结果：
```

2位数为奇数的个数为28
3位数为奇数的个数为224
4位数为奇数的个数为1792
5位数为奇数的个数为14336
6位数为奇数的个数为114688
7位数为奇数的个数为917504
8位数为奇数的个数为7340032
总数为8388608
```
