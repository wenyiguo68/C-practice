题目：打印出所有的"水仙花数"，所谓"水仙花数"是指一个三位数，其各位数字立方和等于该数本身。例如：153是一个"水仙花数"，因为153 = 1的三次方＋5的三次方＋3的三次方。
```c
#include<stdio.h>

int main()
{
	for (int i = 100; i < 1000; ++i)
	{
		int units, tens, huns;
		huns = i / 100;
		tens = i % 100 / 10;
		units = i % 10;
		if (i == huns*huns*huns + tens*tens*tens + units*units*units)
		{
			printf("%d ", i);
		}
	}

	return 0;
}
```
